User Guide

Welcome to the world of Battleship! Ready to gear up and sink ships? Grab your keyboard now and shoot out the winning codes!

Your Goal is to SUNK ALL SHIPS OF YOUR OPPONENT, so how should you wire up your code? Here's how you play:

  1. Write a method that takes in a String argument called "boardStr" that looks like this “-------|-------|-------|-------|-------|-------|-------". Each dash represents a cell. Each "|" represents a row break.
  2. The method should returns a string that looks like “-------|-------|-------|---b---|-------|-------|-------", where "b" is where you want to place the bomb.
  3. Our servers will process your code and return you a string that looks like “-------|-------|---m---|---h---|-------|-------|-------”, where "h" is a hit and "m" is a miss.
  4. The string returned by server will be the new input for the next turn.
